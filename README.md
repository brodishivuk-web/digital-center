# Digital Center - דיגיטל סנטר

חנות אונליין לקורסים דיגיטליים בעולם השיווק בדיגיטל, בנויה עם Next.js, TypeScript ו-Tailwind CSS.

## הרצה מקומית

```bash
npm install
npm run dev
```

האתר יעלה בכתובת [http://localhost:3000](http://localhost:3000).

## שליחת מיילים (אופציונלי)

האתר שולח אישור הזמנה ללקוח והודעה על הזמנה חדשה לחנות. כדי שהמיילים באמת יישלחו יש להגדיר מפתח API של
[Resend](https://resend.com) (יש חבילה חינמית):

1. פותחים חשבון חינמי ב-resend.com ומעתיקים API Key.
2. מוסיפים בפרויקט ב-Vercel (Settings → Environment Variables) את המשתנה `RESEND_API_KEY`.
3. אופציונלי: מגדירים גם `STORE_EMAIL` (כתובת המייל של החנות שתקבל התראות על הזמנות חדשות) ו-`RESEND_FROM_EMAIL`.

ללא הגדרת המפתח, טופס ההזמנה והניוזלטר ימשיכו לעבוד כרגיל - פשוט לא יישלח מייל בפועל.

## סוכן פרסום AI

בכתובת `/ads-agent` יש סוכן שמייצר מודעות מותאמות לכל קורס ומפרסם אותן באתרים שתגדירו:

- **יצירת המודעות** - אם מוגדר משתנה הסביבה `ANTHROPIC_API_KEY` (מפתח מ-[console.anthropic.com](https://console.anthropic.com)),
  הסוכן משתמש ב-Claude כדי לכתוב מודעה ייחודית לכל אתר לפי סוג הפלטפורמה (קבוצת פייסבוק, לוח מודעות, פורום וכו').
  בלי מפתח, הוא משתמש במחולל תבניות מקומי - הכל ממשיך לעבוד.
- **פרסום** - לכל אתר יעד בוחרים איך מפרסמים:
  - `Webhook` - שליחת המודעה כ-JSON לכתובת שהאתר מספק (עובד מצוין עם Zapier/Make לחיבור כמעט לכל אתר).
  - `מייל` - שליחת המודעה למייל של האתר (דרך Resend, דורש `RESEND_API_KEY`).
  - `העתקה ידנית` - הסוכן מכין טקסט מוכן להעתקה ולהדבקה.
- רשימת אתרי היעד נשמרת בדפדפן (localStorage), כך שלא צריך להגדיר אותה מחדש בכל פעם.
